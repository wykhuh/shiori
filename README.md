# Shiori

This is a fork of [Shiori](https://github.com/RadhiFadlillah/shiori)

## Install

```bash
# install dependenicies
go get github.com/shurcooL/vfsgen

```
## Run

``` bash
# recomple assets
go generate

# create binary
go build
```
